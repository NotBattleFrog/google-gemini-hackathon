# Cyber Noir

A murder mystery simulation game powered by Google Gemini AI, built with Godot 4.5.

## Quick Start

1. **Get API Key**: Visit [Google AI Studio](https://makersuite.google.com/app/apikey) to get your Gemini API key
2. **Set API Key**: Press `K` in-game to open the API Key settings panel, or edit `user://settings.cfg` manually
3. **Play**: The game starts directly in the game scene (no menu)

## Features

- AI-powered dynamic conversations between NPCs
- Ghost influence system - whisper messages to characters
- Murder mystery investigation gameplay
- Persistent conversation memory and relationships

## Controls

- `K` - Open/Close API Key Settings Panel
- Mouse - Move ghost character

- Godot 4.5+
- Google Gemini API Key
